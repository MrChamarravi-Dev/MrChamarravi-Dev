## 🤝 Cómo Contribuir y Colaborar

¡Me encanta colaborar y aprender de la comunidad! Si tienes interés en trabajar juntos:

### 📋 Proceso para Contribuir:
1. **Fork** el proyecto que te interese
2. Crea una **rama** con tu feature (`git checkout -b feature/MiAporte`)
3. **Commit** tus cambios (`git commit -m 'Agrego nueva funcionalidad'`)
4. **Push** a la rama (`git push origin feature/MiAporte`)
5. Abre un **Pull Request** con descripción clara
