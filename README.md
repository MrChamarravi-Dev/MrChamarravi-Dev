Actualmente estudio Ingeniería de Sistemas en la **Universidad Industrial de Santander**. Me apasiona el mundo del desarrollo web y aprender nuevas tecnologías a mi propio ritmo. Formo parte del semillero de investigación **MACV** (Machine Learning Analysis and Computer Vision) del grupo **BIVL²ab**. 

Mi visión a largo plazo es realizar una maestría y un doctorado en Ciencias de la Computación, enfocándome en aplicar estas tecnologías en el campo de las Ciencias Biomédicas.

### 🔭 ¿Qué estoy aprendiendo y construyendo ahora?

* 💻 **Desarrollo Web en proceso:** Construyendo y mejorando paso a paso "La 45 Style", experimentando con componentes de registro, estilos modernos (como el *glassmorphism*) y la integración de plataformas de terceros como Firebase y Steam OpenID.
* 🎓 **Proyectos Educativos:** Trabajando en la arquitectura de una plataforma web modular tipo LMS para la preparación del examen **ICFES Saber 11**, explorando cómo implementar hojas de respuestas digitales.
* 📚 **Explorando nuevos mundos:** Aprendiendo sobre modelado 3D para simuladores (creando assets en SketchUp para ETS2) y dominando conceptos de Cálculo II como las sumas de Riemann y las integrales múltiples.
