Formo parte del semillero de investigación **MACV** (Machine Learning Analysis and Computer Vision) del grupo **BIVL²ab**.

### 🔭 ¿Qué estoy haciendo ahora?

* 💻 **Desarrollando "La 45 Style":** Creando componentes de registro seguros, interfaces adaptables con *glassmorphism* e integrando plataformas de terceros (Firebase, Steam OpenID).
* 🎓 **Construyendo un LMS:** Diseñando una plataforma web modular enfocada en la preparación para el examen **ICFES Saber 11**, con hojas de respuestas digitales y simulacros.
* 📚 **Explorando:** El modelado 3D para simuladores de conducción (creando assets en SketchUp para ETS2) y profundizando en conceptos de Cálculo II (sumas de Riemann, integrales múltiples).
